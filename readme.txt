Для решения задачи в качестве основы была выбрана задача о редакционном расстоянии (Расстояние Левенштайна).
Входные данные в файле Input.txt, выходные формируются в output.txt
Если указать в командной строке:
                         spell_checker.exe Input.big Output.big
то программа будет работать с входным файлов Input.big, и выходным файлом - Output.big
Проверял на строках длинной в пятьдесят символов, программа работает около минуты на том примере.
